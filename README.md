# AirQualityIndex

Calculate air quality index and related things.

## Function list
1. Convert Celsius to Fahrenheit.
1. Convert Fahrenheit to Celsius.
1. Calculate Dew Point.
1. Calculate Heat Index.
1. Get effect of Heat Index.
